# 16 - Register formula

The project shows how to register formula if formula is not supported by the Apache Poi

## Dependencies

```
<dependency>
    <groupId>org.apache.poi</groupId>
    <artifactId>poi-ooxml</artifactId>
    <version>3.17</version>
</dependency>
```
